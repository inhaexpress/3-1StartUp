# 웹 기반 AI 챗봇

## 프로젝트 소개
해당 프로젝트는 영화, 게임, 등의 캐릭터의 대사들을 DialoGPT를 사용해 학습시키고, 
만든 챗봇 모델을 React 기반 웹페이지에 통합시켜 사용이 가능하게 합니다. 해당 웹
페이지에서는 여러 캐릭터들과 채팅이 가능하고, 게시판 기능을 이용하여 챗봇 제작을
요청하거나 챗봇을 이용한 경험담을 나눌 수 있습니다.
